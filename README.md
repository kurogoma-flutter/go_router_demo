# go_router_demo

GoRouterパッケージを使い常時表示されるBottom Navigationの実装

### パッケージ
https://pub.dev/packages/go_router

### 該当ドキュメント
https://gorouter.dev/navigator-builder


## Getting Started

```
$ git clone https://github.com/kurogoma-flutter/go_router_demo.git
$ cd go_router_demo
$ flutter pub get
$ flutter run
```
